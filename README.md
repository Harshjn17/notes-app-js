📝 Notes App

A simple and clean Notes App built using HTML, CSS, and JavaScript.
This project demonstrates core frontend concepts like DOM manipulation, event handling, and CRUD operations.

---

🚀 Features

- ➕ Add new notes
- 🗑️ Delete notes
- ✏️ Edit existing notes
- 🔄 Dynamic UI updates (no page reload)

---

🧠 Concepts Used

- JavaScript Array Methods:
  
  - "map()" → update notes
  - "filter()" → delete notes
  - "forEach()" → render notes

- DOM Manipulation

- Event Handling

- State Management (basic)

---

📂 Project Structure

📁 Notes-App
 ├── index.html
 ├── style.css
 └── script.js

---

⚙️ How to Run

1. Clone the repository
2. Open "index.html" in your browser
